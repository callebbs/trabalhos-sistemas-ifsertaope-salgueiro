Sistemas para Internet

O que é o curso?

O curso de Sistemas para Internet capacita profissionais para projetar, desenvolver e instalar bancos de dados, páginas para internet e programas de computador.

Quando conclui a graduação, o tecnólogo em Sistemas para Internet está apto a realizar a configuração e instalação de redes de internet, servidores e páginas de comércio eletrônico, criar bancos de dados, páginas da internet e intranet, além de desenvolver aplicativos para plataformas móveis.

Em resumo, praticamente todo tipo de serviço online fica a cargo deste profissional.

No mercado de trabalho, o tecnólogo em Sistemas para Internet pode atuar em empresas ou órgãos públicos no departamento de Tecnologia da Informação ou oferecendo consultoria para instituições públicas e privadas.
